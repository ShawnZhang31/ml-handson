6. 决策树
====================

与SVM一样，决策树是通用的机器学习算法，可以执行分类和回归任务，甚至是多输出任务。他们是功能强大的算法，能够你和复杂的数据集。

决策树也是随机森林的基本组成部分，是当今最强大的机器学习算法之一。
